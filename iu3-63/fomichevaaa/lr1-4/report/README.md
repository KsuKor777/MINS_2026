# Отчет BMSTU

В этой папке подготовлен автономный шаблон отчета на базе класса `bmstu`.
Файлы класса лежат локально в каталоге `bmstu`, поэтому глобальная установка
пакета `bmstu` не требуется.

## Что заполнить

Перед сборкой отредактируйте [report.tex](/Users/fomicheva_anna/Desktop/МиНС/lr1/report/report.tex:1)
и замените текстовые плейсхолдеры в титульном листе на свои данные.

## Как собрать

Из каталога `report` выполните:

```bash
./build.sh
```

Если LaTeX еще не установлен, нужно установить дистрибутив TeX с поддержкой
`pdflatex` и `biber`. На macOS для этого можно использовать:

```bash
./install_tex_macos.sh
```
