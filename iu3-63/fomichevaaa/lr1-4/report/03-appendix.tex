\begin{appendices}
    \chapter{Инструкция по запуску}

    Для запуска демонстрационного консольного приложения используется команда:

    \begin{verbatim}
python3 main.py
    \end{verbatim}

    Для запуска модульных тестов используется команда:

    \begin{verbatim}
python3 -m unittest discover -s tests -q
    \end{verbatim}

    Для сборки PDF-версии отчета из каталога \texttt{report} используется
    команда:

    \begin{verbatim}
./build.sh
    \end{verbatim}

    Перед сборкой отчета необходимо заполнить поля титульного листа в файле
    \texttt{report.tex}.
\end{appendices}
